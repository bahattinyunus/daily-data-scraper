# Contributing

Teşekkürler! Katkıda bulunmak isterseniz aşağıdaki adımları izleyin:

1. Repoyu fork'layın ve kendi branch'inizde çalışın.
2. Küçük, tek amaçlı commitler yapın ve anlamlı commit mesajları kullanın.
3. Pull request açmadan önce kodunuzu yerel olarak test edin.
4. Büyük değişiklikler için önce bir issue açıp tartışma başlatın.

Basit PR'ler; hata düzeltmeleri veya küçük iyileştirmeler hızlıca merge edilir. Daha büyük özellikler üzerine konuşalım.
